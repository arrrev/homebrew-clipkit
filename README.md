# homebrew-clipkit

Homebrew tap for [ClipKit](https://github.com/arrrev/clipkit) — macOS clipboard manager.

## Install

```bash
brew tap arrrev/clipkit
brew install --cask clipkit
```

## After installing

Grant Accessibility permission so hotkeys work:

**System Settings → Privacy & Security → Accessibility → ClipKit → ON**

Then relaunch ClipKit. Hotkeys: **⌘⌥V** (history) · **⌘⌥T** (transform)

## Uninstall

```bash
brew uninstall --cask clipkit
brew untap arrrev/clipkit
```
